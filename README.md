MultiplyMaster — Тренажёр таблицы умножения на 7 языках
MultiplyMaster — коллекция из семи независимых реализаций интерактивного тренажёра таблицы умножения. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для отработки навыков умножения с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
🎯 Генерация случайных примеров на умножение с настраиваемой сложностью:

Лёгкий: числа от 1 до 5

Средний: числа от 1 до 10

Сложный: числа от 1 до 20

📝 Интерактивный режим — пользователь вводит ответ, программа проверяет и даёт обратную связь.

✅ Подсчёт статистики: количество правильных и неправильных ответов, процент успеха.

⏱️ Замер времени — общее время прохождения теста.

🔄 Возможность повторного запуска без перезагрузки программы.

📊 Подробный отчёт в конце каждой сессии.

🖥️ Интерфейсы:

Командная строка (CLI) для большинства языков (Python, JavaScript, Go, Rust, Java, C#).

Веб-интерфейс для PHP.
